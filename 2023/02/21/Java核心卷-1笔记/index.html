1.1 Java的十一个特性

  • 简单性（省去指针等）
  • 面向对象
  • 分布式
  • 健壮性
  • 安全性
  • 体系结构中立（字节码虚拟机可以使得Java程序可以在多种操作系统上简单运行）
  • 可移植性（基本数据类型的大小及有关运算的行为都有明确的说明，例 int永远为32位）
  • 解释性（Java解释器可以在任何移植了解释器的机器上执行Java字节码）
  • 高性能
  • 多线程
  • 动态性
